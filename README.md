# homebrew-dblite

Homebrew tap for [dblite](https://github.com/BenDavidAaron/dblite)

Installation
------------

To install dblite using Homebrew, run the following command:

```bash
brew tap BenDavidAaron/dblite
brew install dblite
```

Updating this tap
--------

after making a release in the main repo, update the formula by placing the latest release tarball url and sha256 hash in the formula file.
